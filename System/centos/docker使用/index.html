# docker 搭建 ARL

安装 git

  1. pull 镜像

    docker pull tophant/arl

  2. 克隆源码

    git clone https://github.com/TophantTechnology/ARL

  3. 切换到 ARL/docker 目录下查看 config-docker.yaml

    vim ARL/docker/config-docker.yaml

  4. docker volume create --name=arl_db

  5. docker-compose up -d

  6. 查看是否正常运行

    docker-compose ps

  7. 如何更新 ARL

    cd /ARL

    git pull

    cd docker

    docker-compose pull

    docker-compose up -d

  8. 如何打开

    cd ARL/docker

    docker-compose up -d

    docker-compose ps

    https://127.0.0.1:5003

    默认用户名密码 admin/arlpass
