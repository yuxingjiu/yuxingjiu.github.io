Docker安装

  1. 使用yum源下载docker安装组件
yum install --downloadonly --downloaddir=/root/dockerRPM docker
  1. 创建离线包关系
createrepo /root/dockerRPM
  1. 配置yum源
rm -rf /etc/yum.repos.d/*
vim test.repo

[test]
name=test
baseurl=file:///root/dockerRPM
enable=1
gpgcheck=0
  1. 安装docker
yum install docker
  1. 开启服务查版本
systemctl start docker
docker version
  1. 设置国内镜像仓库

访问 https://cr.console.aliyun.com/ 获取加速器

修改配置文件

/etc/docker/daemon.json (如果该文件不存在，则创建)

{
     "registry-mirrors": [
        "https://fs4tjg48.mirror.aliyuncs.com"
     ]
}
  1. 重新加载
systemctl daemon-reload
systemctl restart docker.service
  1. 安装docker-compose
pip -v
yum install python3-pip
pip3 install docker-compose

Docker操作

1、搜索镜像

docker search 镜像名
参数说明
NAME镜像的名字
DESCRIPTION镜像的描述
STARTS标识喜欢
OFFICIAL是否docker官方发布
AUTOMATED自动搭建

2、获取镜像

docker pull 镜像名字

3、容器的操作

查看镜像

docker images

docker ps // 查看正在运行的容器

docker ps -a // 查看所有的容器

docker ps -l // 查看最近一次运行的容器

docker top 容器名 // 查看web应用程序容器的进程

启动镜像，变为容器

参数描述
-itd表示交互式守护进程
–name给原先的镜像名重新起名字
-p将容器的80端口映射到本地的801端口上

注意 : docker操作中的ID号可以在创建时 设置别名，通过别名去代替ID也是可以的

比如，创建了一个名为web的容器，ID号5a1883ba8852，在对容器操作时，就可以用web代替ID号

docker run -itd --name 别名 镜像文件名(或者镜像ID号，一般输2位即可) /bin/bash

docker run -itd --name web 21 /bin/bash

docker run -itd --name web docker.io/nginx /bin/bash

docker run -itd --name web -p 801:80 e21

关闭容器

docker stop 对应ID

进入容器

docker exec -it 对应ID /bin/bash

删除容器

docker rm 对应ID

复制文件或目录

  1. 通过 docker cp 去复制
docker cp 目录或文件 ID号:容器目录

docker cp /var/www/html/index.html ID号:/usr/share/nginx/html/index.html
  1. 通过创建新的容器时对目录进行映射-
docker run -itd --name web2 -p 802:80 -v /nginx-web/:/usr/share/nginx/html 容器ID

docker run -itd --name web2 -p 802:80 -v /nginx-web/:/usr/share/nginx/html 99e3

docker搭建ARL

安装git

yum install git

  1. pull镜像

    docker pull tophant/arl

  2. 克隆源码

    git clone https://github.com/TophantTechnology/ARL

  3. 切换到ARL/docker目录下查看config-docker.yaml

    vim ARL/docker/config-docker.yaml

  4. docker volume create –name=arl_db

  5. docker-compose up -d

  6. 查看是否正常运行

    docker-compose ps

  7. 如何更新ARL

    cd /ARL

    git pull

    cd docker

    docker-compose pull

    docker-compose up -d

  8. 如何打开

    cd ARL/docker

    docker-compose up -d

    docker-compose ps

    https://127.0.0.1:5003

    默认用户名密码admin/arlpass
