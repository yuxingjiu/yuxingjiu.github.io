# 端口扫描

# 目录扫描

没有什么可用的信息，查看一下 web 界面

80 端口没有什么可用的信息，查看一下 21 端口，看能否通过默认账号登录 (anonymous / 随便邮箱或空密码)

登录失败，只能尝试爆破 (hydra)

# ftp 爆破

由于很长时间都没有爆破成功，返回去查看这个靶机介绍，显示

通过介绍可以尝试账户名是 <mark style="background: #FF5582A6;">chili</mark>

最终得出账号密码 (chili /a1b2c3d4)

显示 230，证明登陆成功

可以切换到网站目录，查看权限，发现 .nano 目录是可读可写可执行的，尝试写入一句话木马

分别上传到目标

当使用浏览器进行访问时，发现为空白页面，再次尝试一下反弹 shell

反弹 shell 可以网上找找

这里需要修改 IP 和端口，之后使用 nc (nc -lvvp 1234) 开始监听

查看之后发现没有权限，那么之前访问不到也是有可能没有权限才访问不到

拿蚁剑测试看能否连接成功

OK，成功连接，尝试使用蚁剑去反弹 shell，发现失败

看来只能使用之前的 shell.php 来进行连接

成功反弹，使用 python 代码将终端格式化

查看一下能否提权

发现并没有，查看一下一些系统信息

发现这个 passwd 文件尽然开放着写权限

通过 openssl 生成密码直接添加用户

# openssl 添加用户

根据 passwd 目录生成一行数据

成功登录
