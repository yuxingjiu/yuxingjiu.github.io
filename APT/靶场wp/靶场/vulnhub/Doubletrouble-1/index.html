目录扫描

dirsearch -u http://192.168.50.152

重新安装发现安装失败（密码错误）

根据页面信息可以找到该网站用了qdPM 9.1 ，该版本存在命令执行漏洞

漏洞利用

搜索关于 dPM 9.1 的漏洞

查看源码发现需要登录的邮箱和密码，但是前面的账号密码不正确，继续在/secret下面发现一张图片

图片隐写

使用 steghide 查看是否有隐写

发现需要密码，肯定存在信息

https://github.com/RickdeJager/stegseek/releases
apt install ./stegseek_0.6-1.deb

重新输入密码查看

成功登录
