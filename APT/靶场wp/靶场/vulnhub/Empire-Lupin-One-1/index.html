端口扫描

发现开放22端口和80端口，22端口显示存在历史ssh秘钥，80端口显示有robotx.txt信息

目录扫描

利用fuzz测试页面

wfuzz -c -z file,/usr/share/wordlists/wfuzz/general/big.txt --hc 404,403 http://192.168.50.149/\~FUZZ

可以或得到的信息有 icex64 是一个用户，而且这个 ~secret 是一个目录，目录下面存放着解密ssh文件

wfuzz -c -z file,/usr/share/wordlists/dirbuster/directory-list-2.3-medium.txt --sc 200 http://192.168.50.149/~secret/.FUZZ.txt

ssh连接

将密文保存到文件中，使用文件进行登录

ssh icex64@192.168.50.149 -i key

发现使用密文还得输入密码

使用 ssh2john 将其装换为可识别的hash，然后使用john工具对其破解

/usr/share/john/ssh2john.py key > hash


john --wordlist=/usr/share/wordlists/fasttrack.txt hash

得出密文 P@55w0rd!
