# 文件操作

文件读取的操作，前提条件

  1. 读取权限是否允许 select @@secure_file_priv
  2. 直到目标机器的绝对路径
  3. 读文件必须小于 max_allowed_packet 所设置的值 ``show VARIABLES like '% max_allowed_packet%';`
  4. 写文件目标目录是否可写

# 二次排序注入

二次排序注入也成为存储型的注入，就是将可能导致 sql 注入的字符先存入到数据库中，当再次调用这个恶意构造的字符时，就可以出发 sql 注入。
