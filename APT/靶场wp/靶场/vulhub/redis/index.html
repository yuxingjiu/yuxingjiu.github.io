# 注意事项

  1. webshell 写入可以参考上面的
  2. 反弹 shell 时写入的任务计划名称是以用户名命名的，想要获得谁的权限就把名字改成谁 [config set dbfilename 名字]
  3. ssh 写公钥时保存的名字不需要加引号 [config set dbfilename autorized_keys] 这样就行了，不要给名字加引号，不然 ssh 连接失败
