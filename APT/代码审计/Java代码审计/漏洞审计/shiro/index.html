shiro

shiro550序列化

该漏洞版本 <= shiro1.2.4

环境搭建

  1. 下载shiro1.2.4

https://archive.apache.org/dist/shiro/1.2.4/shiro-root-1.2.4-source-release.zip

  1. 使用idea导入该项目（ new object -> 选择下载的shiro1.2.4 ）

  2. 修改smaples/web/pom.xml​文件

​
    <properties>
        <project.build.sourceEncoding>UTF-8</project.build.sourceEncoding>
        <maven.compiler.source>1.6</maven.compiler.source>
        <maven.compiler.target>1.6</maven.compiler.target>
    </properties>

        <groupId>org.apache.maven.plugins</groupId>
        <artifactId>maven-war-plugin</artifactId>
        <version>3.2.2</version>
​
        <dependency>
            <groupId>javax.servlet</groupId>
            <artifactId>jstl</artifactId>
            <version>1.2</version>
            <scope>runtime</scope>
        </dependency>
​
        <dependency>
            <groupId>commons-collections</groupId>
            <artifactId>commons-collections</artifactId>
            <version>3.2.1</version>
        </dependency>
​
  1. 修改samples\pom.xml​文件 （ 375到394代码进行注释 ）
​
  1. 使用Maven重新载入（ 分别选中两个被修改的pom.xml文件右键同样操作 ）
​
  1. 选择修改设置，使用Tomcat加载 ( 自行下载Tomcat )
​
  1. 配置Tomcat，选择war包
​ ​

保存配置后点击运行

​
