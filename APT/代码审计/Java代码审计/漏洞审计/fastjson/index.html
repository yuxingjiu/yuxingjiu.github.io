Fastjson 基础

fastjson 除了基础的 json 功能外，还可以将 json 字符串解析成 javabean（set、get 等）

将 json 字符串解析成一个对象，还会调用它的构造函数、set 方法

  1. 先按照字符串进行解析
  2. 解析过程发现存在 @type
  3. 存在 @type 需要按照 java 对象解析
  4. 按照该 java 对象解析，对该类需要一个反序列化器
  5. 为了获取该反序列化器，在解析的类中定义一个 getMap 方法，不需要 set 方法
  6. 之后调用它对应的反序列化方法
    1. 首先遍历所有的字段
    2. 如果存在该字段对应的反序列化器，构造的时候，默认就有一个
    3. 调用构造函数后赋值（要么反射，要么 set 方法，这里是 set 方法）

存在问题

  1. 用反序列化器解析的时候会调用它的构造方法
  2. 赋值的时候会调用它的 set 方法
  3. JSON.ParseObject () 最后会调用一个 toJSON 方法，会调用 get 方法
  4. 如果被解析的变量满足 getOnly 格式（就只有 get 方法没有 set 方法，且返回值是固定的那几种类型之一，像 map 等）也会调用

和原生反序列化的区别

  1. fastjson 不需要实现serializable
  2. 变量不需要不是 transient，变量有对应的 setter 或者是 public
  3. setter、getter 不是 readObject
  4. sink 反射、动态类加载
