# Java 反射

# 反射介绍

**Java 反射 (**​ Reflection ​) 是 Java 非常重要的动态特性，通过使用反射我们不仅可以获取到任何类的成员方法 ( Methods ​)、成员变量 ( Fields ​)、构造方法 ( Constructors ​) 等信息，还可以动态创建 Java 类实例、调用任意的类方法、修改任意的类成员变量值等。Java 反射机制是 Java 语言的动态性的重要体现，也是 Java 的各种框架底层实现的灵魂

​

在 Java 中看到的绝大部分成员，其实都可以称之为对象（ 除了普通数据类型和静态成员 ）​ public class Student{} ​

类也是对象，类是 java.lang.Class ​类的实例对象，比如 Student 的类，这个类本身就是一个对象 Class 类的实例对象

new Student().getClass().getClass() ​

也就是说 new Student().getClass() ​获取的是 class cmder.CmdSerialize ​

而 new Student().getClass().getClass() ​就是获取的是 class java.lang.Class ​
