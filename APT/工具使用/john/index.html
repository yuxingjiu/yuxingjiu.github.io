# john

john 是一款经典的码破解工具，破解 hash 密码

# john 的模式

single 模式

不实用字典来爆破，使用用户的用户名或用户登陆的服务等信息，通过变形来计算密码（单词变形）

参数描述
--list=formats查看支持密码的格式 [算法]
--show --format = 密文类型 密文查看已经被破解出来的密码
--wordlist指定字典
--format指定 hash 算法，格式前面存 Raw 表示 hash 没有使用加盐，加盐是 salt

john 默认支持自动检测（分析）密文，但是不准确，可以通过两个工具帮忙分析

  1. 在线网站 http://gchp.github.io/CyberChef
  2. kali 自带的工具 hash-identifier

# linux 口令

因为 linux 的密文存储是影子账号，及账号 /etc/passwd 密码 /etc/shadow ，想要破解，先对其进行结合

# ssh 口令破解

openssl 生成

ssh 密钥用于 ssh 方式登陆设备，在允许用户使用 ssh 密钥登陆之前，将需要啊一个用于身份验证的密码，二 john 就是破解的这个密码

但是 ssh 公钥 john 是无法识别的，需要通过工具 （ssh2john）将密文转为 john 可识别的类型
