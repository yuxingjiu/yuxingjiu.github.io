# Access 数据库

Access 数据库没有系统自带库，所以在每个查询语句的后面都要带表名

当 GET 传入 id=1 和 cookie 传入 id=2 时，页面会显示 id=2 的内容，这里是因为 cookie 的优先级要高于 GET

# 偏移注入

  1. 当不知道表名不知道字段名的 情况下使用偏移注入
  2. 因为 SQL 注入时查询网站写好的数据表，想要注入别的数据表必须字段数相同，或者想要查询的数据表的字段小于当前查询的数据表的字段
  3. 表名.* 代表这个数据表里面的所有数据

order by 12 => 有 12 个字段

想要注入的 admin 表有 6 个字段

通过调整前后的参数来找到显错点（好比这里的显错点为 3，就可以进行变化）
