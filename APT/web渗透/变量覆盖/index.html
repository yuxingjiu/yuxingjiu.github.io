# 变量覆盖

变量覆盖指的是可以利用我们的传参值替换程序原有的变量值

# $$ 使用不当

这个代码会接收我们的 GET、POST、Cookie 参数

将这个接收来的参数依次放入 $$_request as $_key=>$_value 这是个数组解析

实际上就是键值分离， $a=1 是一个固定的值，但是因为 $$_key 的原因
当我们传参 a=2 时 $$_key=addslashes($_value) 就变成了 $a=2

# extract()

数组中将变量导入到当前的符号

# parse_str () 函数

将字符串解析成变量

输出了 zkaq 和 60

那么 parse_str("name=Bill&age=60") 相当于完成了 $name ='zkaq'和$age ='60'
