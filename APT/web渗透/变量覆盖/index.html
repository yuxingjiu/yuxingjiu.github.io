变量覆盖

变量覆盖指的是可以利用我们的传参值替换程序原有的变量值

$$使用不当

<?php
    $a = 1;
    foreach(array('_COOKIE','_POST','_GET') as $_request) {
    foreach($$_request as $_key=>$_value)
    {$$_key=addslashes($_value);}}
    echo $a;
?>

这个代码会接收我们的GET、POST、Cookie参数

将这个接收来的参数依次放入$$_request as $_key=>$_value这是个数组解析

实际上就是键值分离，$a=1是一个固定的值，但是因为$$_key的原因
当我们传参a=2时$$_key=addslashes($_value)就变成了$a=2

extract()

数组中将变量导入到当前的符号

<?php
    $a = "1";
    $my_array = array("a" => "Cat","b" => "Dog", "c" => "Horse");
    extract($my_array);
    echo "\$a = $a; \$b = $b; \$c = $c"; // $a = Cat; $b = Dog; $c = Horse
?>

parse_str()函数

将字符串解析成变量

<?php
    parse_str("name=zkaq&&age=60");
    echo $name."<br>";
    echo $age;
?>

输出了zkaq和60

那么parse_str("name=Bill&age=60") 相当于完成了$name ='zkaq'和$age ='60'
