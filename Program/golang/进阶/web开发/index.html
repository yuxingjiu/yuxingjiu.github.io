消息头第一行由三个以空格分隔的元素组成，分别为 HTTP 方法、请求的 URL、使用的 HTTP 版本

GET、POST、HEAD、TRACE、OPTIONS、PUT、DELETE、CONNECT

应用开发

http 包提供了 HTTP 服务器和客户端的开发接口，内置 web 服务器

# 服务端

针对 web 服务端开发流程为

  1. 处理函数 func(http.ResponseWriter, *http.Request){}
  2. 绑定 URL 关系 http.HandleFunc() 、 http.Handle()
  3. 启动 web 服务器 http.ListenAndServe()

绑定 URL 关系可以通过很多函数设置处理器去处理

# 处理器创建

# http.HandleFunc()

  • 处理器函数

该函数接收两个参数，一个字符串（该字符串是我们指示服务器去寻找的 URL 路径模式）和一个实际处理请求的函数

而这个函数（自定义或内联）需要接收两个参数

  1. 第一个是 http.ResponWriter [用于写入对请求的响应]
  2. 第二个是指向 http.Request 的指针 [它将允许我们从传入的请求中读取信息]

不是在主函数中调用 manage，而指示告诉 HTTP 服务器对 /hello 的所有请求都是由 manage () 函数处理

该函数既可以为匿名内联函数

还可以是自定义函数
