# 接口

  1. 接口是自定义类型，是对其他类型行为的抽象
  2. 接口定义使用 interface 标识，声明了一系列的函数签名（函数名、参数、返回值），定义接口时可以指定接口名称，在后续声明接口变量时使用
  3. 如果想要将一个结构体类型变量赋值给接口变量，这个结构体对象 必须实现接口中的行为抽象 （方法）
  4. 接口对象不能访问对应结构体的属性，但是可以访问其方法
  5. 实现面向对象（结构体 => 封装、匿名组合 => 继承、接口 => 多态）

# 断言

很多情况下，将结构体转为接口后，只能实现接口的方法，无法实现原先结构体的一些方法或者参数，这里便可使用断言将接口对象转为结构体对象

obj, ok := 接口变量.(结构体对象)

# switch 替换 if 断言

由于 if 断言时可能存在很多选择，可以使用 switch 替代 if 作判断，但是首先得获取其对应的类型

接口变量.(type) // 获取对应的类型

.(type) 只能用在 switch case 语句中

# 反射

反射是指在运行时动态的访问和修改任意类型对象的结构和成员，在 go 语言提供 reflect包 提供反射的功能，每一个变量都有两个属性 => 类型（Type）和值（Value）

# Type

reflect.Type 是一个接口类型，用于获取变量类型的信息，可以通过 reflect.TypeOf() 获取某个变量的类型信息
