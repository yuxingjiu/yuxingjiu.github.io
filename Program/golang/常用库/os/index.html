# os.OpenFile()

按照指定格式打开文件

格式介绍
O_RDONLY打开只读文件
O_WRONLY打开只写文件
O_RDWR打开既可以读取又可以写入文件
O_APPEND写入文件时将数据追加到文件尾部
O_CREATE如果文件不存在则创建一个新的文件
O_EXCL文件必须不存在，然后会创建一个新的文件
O_SYNC打开同步 I/O
O_TRUNC文件打开时可以截断

# os.Chtimes()

修改文件访问和修改时间

package main

import (
   "os"
   "time"
)

func main() {
   _, _ = os.Create("123.txt")
   parse, _ := time.Parse("2006/01/02", "1998/12/11")

   err := os.Chtimes("123.txt", parse, parse)
   if err != nil {
      return
   }

}

# os.StartProcess()

开启新进程

StartProcess () 共有三个参数，第一个是软件路径，第二个是执行软件及参数（如果没有就为空），第三个是进程属性

// 开始一个新进程

if err != nil {
	fmt.Println(err)
}
