go help

直接在终端输入 go help 即可显示所有go命令以及响应命令功能简介

build ：编译包和依赖
clean ：移除对象文件
doc ：显示包或者符号的文档
env ：打印go的环境信息
bug ：启动错误报告
fix ：运行go tool fix
fmt ：运行gofmt进行格式化
generate ：从processing source生成go文件
get ：下载并安装包和依赖
install ：编译并安装包和依赖
list ：列出包
run ：编译并运行go程序
test ：运行测试
tool ：运行go提供的工具
version ：显示go的版本
vet ：运行go tool vet

go build

编译程序，会在当前目录生成可执行文件

go build -x xxx.go可以查看其编译过程中调用了什么指令

go build -o 1.exe xxx.go 指定输出的文件名字

go run

像执行脚本代码一样执行go脚本

go env

go get

通过官方仓库找到向你要用的第三方包，复制链接地址

进行下载之前必须得将项目进行初始化

go mod init 项目名
go get github.com/dop251/goja

go install

分为两步，先执行编译，得到一个可执行文件，之后将可执行文件呢拷贝到GOPATH下的bin目录

go mod

  1. 初始化模块
go mod init 名称
  1. 移除不依赖的包
go mod tidy
  1. 获取模块列表
go list -m -json all

交叉编译

  1. Go支持跨平台编译
  2. 在Windows平台编译一个能在Linux平台执行的可执行文件

Mac

CGO_ENABLED=0 
GOOS=linux 
GOARCH=amd64 
go build main.go

CGO_ENABLED=0 
GOOS=windows 
GOARCH=amd64 
go build main.go

Linux

CGO_ENABLED=0 
GOOS=darwin 
GOARCH=amd64 
go build main.go

CGO_ENABLED=0 
GOOS=windows 
GOARCH=amd64 
go build main.go

Windows

SET CGO_ENABLED=0
SET GOOS=darwin
SET GOARCH=amd64
go build main.go

SET CGO_ENABLED=0
SET GOOS=linux
SET GOARCH=amd64
go build main.go
