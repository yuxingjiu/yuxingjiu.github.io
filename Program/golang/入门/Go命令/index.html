# go help

直接在终端输入 go help 即可显示所有 go 命令以及响应命令功能简介

build ：编译包和依赖
clean ：移除对象文件
doc ：显示包或者符号的文档
env ：打印 go 的环境信息
bug ：启动错误报告
fix ：运行 go tool fix
fmt ：运行 gofmt 进行格式化
generate ：从 processing source 生成 go 文件
get ：下载并安装包和依赖
install ：编译并安装包和依赖
list ：列出包
run ：编译并运行 go 程序
test ：运行测试
tool ：运行 go 提供的工具
version ：显示 go 的版本
vet ：运行 go tool vet

# go build

编译程序，会在当前目录生成可执行文件

go build -x xxx.go 可以查看其编译过程中调用了什么指令

go build -o 1.exe xxx.go 指定输出的文件名字

# go run

像执行脚本代码一样执行 go 脚本

# go env

# go get

通过官方仓库找到向你要用的第三方包，复制链接地址

进行下载之前必须得将项目进行初始化

# go install

分为两步，先执行编译，得到一个可执行文件，之后将可执行文件呢拷贝到 GOPATH 下的 bin 目录

# go mod

  1. 初始化模块
  1. 移除不依赖的包
  1. 获取模块列表
