变量

变量的类型列表

变量的功能数存储数据，不同的变量保存的数据类型可能会不一样

经过半个多世纪的发展，编程语言已经基本形成了一套固定的类型

常见变量的数据类型有 ：整形、浮点型、布尔型、字符串、字符

Go语言中的每一个变量都有自己的类型，并且变量必须经过声明才能开始使用

变量的声明

Go语言中的变量需要声明后才能使用，同一作用于内不支持重复声明，并且Go语言的非全局变量声明后必须使用，不使用就无法编译成功

标准声明

Go语言的变量声明格式为

var 变量名 变量类型

变量声明以关键字var开头，变量类型放在变量的后面，行尾无需分号

单独声明

var name string

批量声明

var(
    name string // ""
    age int     // 0
    isNull bool // false
)

变量的初始化

初始化后赋值

// 初始化变量、数据类型
var (
    name1   string
    age1    int
    isPass1 bool
)
// 赋值
name1 = "MengPeiYao"
age1 = 22
isPass1 = true

初始化变量、类型、值

// 初始化变量、数据类型、初始值
var (
    name2   string = "mpy"
    age2    int    = 18
    isPass2 bool   = false
)

初始化变量、值

  • 根据值判断数据类型
// 初始化变量、初始值【根据值来判断数据类型】
var (
    name3   = "mpy"
    age3    = 21
    isPass3 = true
)

短变量声明

使用:=赋值【m:=123】

// 初始化变量
name4 := "mengpeiyao"
age4 := 23
isPass4 := true

匿名变量声明

在使用多重赋值时，如果想要忽略某个值，可以使用匿名变量（anonymous variable）

匿名变量用一个下划线 _ 表示

func foo() (int, string) {
    return 10, "Q1mi"
}
func main() {
    x, _ := foo()
    _, y := foo()
    fmt.Println("x=", x) // x=10
    fmt.Println("y=", y) // y=Q1mi
}

匿名变量不占用命名空间，不会分配内存，所以匿名变量之间不存在重复声明。 (在Lua等编程语言里，匿名变量也被叫做哑元变量)

注意事项：

  1. 函数外的每个语句都必须以关键字开始（var、const、func等）
  2. :=不能使用在函数外
  3. _多用于占位，表示忽略值

变量的作用域

小范围可以引用大范围的变量，但是大范围无法引用小范围的变量

package main

import "fmt"

var num int = 100

func main() {

    {
        num = 200
    }

    fmt.Println(num)
}

此时num的值为200

常量

const

相对于变量，常量是恒定不变的量

多用于程序运行期间不会改变的那些值

常量的声明和变量声明类似，只是关键字不一样，是const，常量在定义的时候就得赋初值

单个声明

// 单个声明
const pi = 3.1415926

多个声明

// 批量声明
const (
    status  = 200
    notFond = 404
    isError = 502
)

// 如果批量声明常量值，如果某一行声明后没有赋值，则默认和上一行的一致
const (
    n1 = 100 // 100
    n2		 // 100
    n3 = 200 // 200
    n4		 // 200
)

iota

常量计数器

iota在const关键字出现时被重置为0

const中每增加一行常量声明将是iota计数一次【iota可理解为const语句快中的索引】，使用iota能简化定义，在定义枚举时有用

const (
    n1 = iota   // 0
    n2          // 1
    n3          // 2
    n4          // 3
) // 最终输出0,1,2,3

使用_跳过某些值

const (
    a1 = iota   // 0
    a2          // 1
    _           // 2
    a3          // 3
    a4          // 4
) // 最终输出0,1,3,4

插队

const (
    b1 = iota
    b2 = 100
    b3 = iota // 这里不重置为0是因为这个const()是一个整体，上面已经被重置了，下面就不会被重置
    b4
) // 最终输出 0,100,2,3

多个常量声明在一行

const (
    j1, j2 = iota + 1, iota + 2 // iota=0 j1=1 j2=2
    j3, j4 = iota + 1, iota + 2 // iota=1 j3=2 j4=3
) // 最终输出1,2,2,3

定义数量集

const (
    _ = iota
    KB = 1 << (10 * iota)
    MB = 1 << (10 * iota)
    GB = 1 << (10 * iota)
    TB = 1 << (10 * iota)
    PB = 1 << (10 * iota)
)
/*
KB: 1024
MB: 1048576
GB: 1073741824
TB: 1099511627776
PB: 1125899906842624
*/
