包的操作

发表于 2023-01-12
  • 余星酒
首页 编程 Golang 基础

包是多个Go源码的集合，是一种高级的代码服用方案，Go语言为我们提供了很多内置包fmt、os、io等

包的定义

在包里面定义的标识符[变量名、函数名、结构体、接口等等]如果是小写字母开头，说明这些标识符只在这个包里面可以使用，当别人导入时是无法使用的，是私有的，如果外部想要使用的话，就必须让标识符首字母大写 可以根据自己的需要创建自己的包，一个包可以简单理解为一个存放.go文件的文件夹，该文件夹下面的所有go文件都要在代码的第一行添加如下代码，声明该文件归属的包

package 包名
  1. 一个文件夹下面只能有一个包，同样一个包的文件不能在多个文件夹下
  2. 包名可以不喝文件夹的名字一样，包名不能包含-符号
  3. 包名为main的包为应用程序的入口包，编译时不包含main包的源代码时不会得到可执行文件

包的导入

需要在代码中引用其他包的内容，需要使用import关键字导入使用的包

import "包的路径"
  1. import导入语句通常放在文件开头包声明语句的下面
  2. 导入的包名需要使用双引号包裹起来
  3. 包名是从$GOPATH/src后开始计算的，使用/进行路径分隔
  4. Go语言中禁止循环导入包

单行导入

import "包1"
import "包2"

多行导入

import {
    "包1"
    "包2"
}

自定义包名

import calc "study.com/study/100calc" // 别名 "包的路径"

import {
    "fmt"

    calc "study.com/study/100calc" // 别名 "包的路径"
}

匿名导入

如果只是希望导入包，而不使用包内部的数据时，可以使用匿名导入包

import _ "包的路径"

匿名导入的包和其他包一样会与其他方式导入的包一样都会被编译到可执行文件中
