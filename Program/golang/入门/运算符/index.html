位运算符

运算符描述
&参与运算的两数各对应的二进制位相与(两位均为1才为1)
|参与运算的两数各对应的二进制位相或(有一位为1就为1)
^参与运算的两数各对应的二进制位相异或(两位不一样就为1)
<<左移n位就是乘以2的n次方(1<<2 ==> 4)
>>右移n位就是除以2的n次方(1>>2 ==> 0)
package main

import "fmt"

// 位运算

func main() {
    num := 1
    fmt.Println(num << 2)
    fmt.Println(num >> 2)
}
