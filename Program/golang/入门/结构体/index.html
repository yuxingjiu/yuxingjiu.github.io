结构体

发表于 2023-01-18
  • 余星酒
首页 编程 Golang 基础

Go语言中没有类的概念，也不支持类的继承等面向对象的概念，Go语言中通过结构体的内嵌在配合接口比面向对象具有更高的扩展性和灵活性

类型别名和自定义类型

自定义类型

Go语言中有一些基本的数据类型，如string、整形、浮点型、布尔等数据类型

Go语言可以通过type关键字来定义自定义类型

自定义类型是定义了一个全新的类型，可以基于内置的基本类型定义，也可以通过struct定义

// 将MyInt定义为int类型
type MyInt int

通过type关键字的定义，MyInt就是一种全新的类型，它具有int的特性

类型别名

类型别名与类型定义表面上看只有一个等号的差异

// 类型定义
type NewInt int

// 类型别名
type MyInt = int

func main() {
    var a NewInt = 100
    var b MyInt = 100
    fmt.Printf("%T %d\n", a, a) // main.NewInt 100
    fmt.Printf("%T %d\n", b, b) // int 100
}

区别

  1. 类型定义相当于定义了一个全新的类型，与之前的类型不同，但是类型别名并没有定义一个新的类型，而是使用了一个别名来替代之前的类型
  2. 类型别名只会在代码中存在，编译完成之后并不会存在该别名
  3. 因为类型别名和原来的类型是一致的，所以原来类型所拥有的方法，类型别名中也可以调用，但是如果是重新定义的一个类型，那么不可以调用之前的任何方法

结构体

Go语言中的基础数据类型可以表示一些事物的基本属性，但是当我们想表达一个事物的全部成分或属性时，这时候再用单一的基本数据类型明显就无法满足需求了，Go语言提供乐一中自定义数据类型可以封装多个基本数据类型，这种数据类型叫结构体[struct]也就是我们可以通过struct来定义自己的类型了

Go语言通过struct来实现面向对象

结构体的定义

使用type和struct关键字来定义结构体

type 类型名 struct {
    字段名 字段类型
    字段名 字段类型
    …………
}

类型名 ：表示自定义结构体的名称，在同一个包内不能重复

字段名 ：表示结构体字段名，结构体字段名必须唯一

字段类型 ：表示结构体字段的具体类型

定义一个Person[人]结构体

type person struct{
    name string
    age int
    city string
    hobby []string
}

这样就拥有了一个person的自定义类型，可以通过这个结构体很方便的在程序中表示和存储人信息

语言内置的基础数据类型是用来描述一个值的，而结构体是用来描述一组值的【姓名、年龄、爱好等】

结构体实例化

package main

import "fmt"

// 结构体

type person struct {
    name   string
    age    int
    class string
    aihao  []string
}

func main() {
    // 声明person类型，通过字段名去赋值
    var xiaoming person
    xiaoming.name = "xiaoming"
    xiaoming.age = 18
    xiaoming.gender = "shuxue"
    xiaoming.aihao = []string{"dalanqiu", "tizhuqiu"}
    fmt.Println(xiaoming)
    // 访问person字段的值
    fmt.Println(xiaoming.aihao)

}

通过.来访问结构体的字段【成员变量】【xiaoming.age、xiaoming.name等】

匿名结构体

在定义一些临时数据结构等场景下还可以使用匿名结构体

package main

import "fmt"

// 匿名结构体

func main() {
    var s struct {
        name string
        age  int
        city string
    }
    s.name = "xiaozhang"
    s.age = 21
    s.city = "taiyuan"
    fmt.Println(s)

}

指针类型结构体

通过使用new关键字对结构体进行实例化，得到的是结构体的地址

var p2 = new(person)


fmt.Printf("%T \n", p2) 
// *main.person
fmt.Printf("p2=%#v \n", p2) 
// p2=&main.person{name:"", age:0, class:"", aihao:[]string(nil)}

从打印结构可以看出p2是一个结构体的指针

需要注意的是Go语言中支持结构体指针直接使用.来访问结构体的成员

结构体初始化

通过.来对成员进行初始化

var p2 = new(person)
p2.name = "小明"
p2.age = 22
p2.class = "数学"
p2.aihao = []string{"看电视", "玩游戏", "打羽毛球"}
fmt.Printf("p2=%#v\n",p2)
 // p2=&main.person{name:"小明", age:22, class:"数学", aihao:[]string{"看电视", "玩游戏", "打羽毛球"}}

key-value格式初始化

var p3 = person{
    name : "xiaoming",
    age : 22,
    class : "语文",
    aihao : []string{"看电视", "玩电脑"},
}
fmt.Println("%#v\n", p3) 
// main.person{name:"xiaoming", age:22, class:"语文", aihao:[]string{"看电视", "玩电脑"}}

使用值列表的形式进行初始化

【根据结构体的初始化顺序来进行赋初值】，值的顺序和结构体初始化的顺序保持一致

var p4 = person{
    "xiaoming",
    22,
    "语文",
    []string{"看电视", "玩电脑"},
}
fmt.Println("%#v\n", p3) // main.person{name:"xiaoming", age:22, class:"语文", aihao:[]string{"看电视", "玩电脑"}}

结构体的匿名字段

结构体允许其成员字段在声明时没有字段名而只有类型，这种没有名字的字段就称为匿名字段

package main

import "fmt"

// 结构体匿名字段

type person struct {
    string
    int
}

func main() {
    p := person{
        "小明",
        12,
    }
    fmt.Println(p) // {小明 12}
}

匿名字段默认采用类型名作为字段名，结构体要求字段名称必须唯一，因此一个结构体中同种类型的匿名字段只能有一个

嵌套结构体

一个结构体中包含着另一个结构体

package main

import "fmt"

// 结构体嵌套

type person struct {
    name  string
    age   int
    addrs addr
}
type addr struct {
    city    string
    address string
}

func main() {
    p := person{
        name: "小王",
        age:  21,
        addrs: addr{
            city:    "杭州",
            address: "浙江省杭州市",
        },
    }
    fmt.Println(p) // {小王 21 {杭州 浙江省杭州市}}
}

结构体模拟继承

package main

import "fmt"

// 结构体模拟继承
type animal struct {
    name string
}

func (a animal) move() {
    fmt.Printf("%s会动\n", a.name)
}

type dog struct {
    waihao string
    animal // 这里就相当于是继承
}

func (d dog) jiao() {
    fmt.Printf("%s[%s]在叫\n", d.animal.name, d.waihao)
}

func main() {
    d1 := dog{
        animal: animal{
            name: "狗",
        },
        waihao: "大黄",
    }
    fmt.Println(d1) // {大黄 {狗}}
    d1.jiao()       // 狗[大黄]在叫
    d1.move()       // 狗会动
}

结构体与JSON

json是一种轻量级的数据交换格式

注意事项 ：

结构体中的变量小写状态下只能在该包中使用，无法在其他包中使用，必须大写

想要指定小写格式输出，可以在结构体变量后面加上反引号``在里面加入对应的格式[由键值组成，中间用冒号隔开，没空格，多种格式用空格分开]

json:”name” db:”database_name” ini:”config_name”

结构体转JSON

json.Marshal(要转换的数据) 返回value和err
package main

import (
    "encoding/json"
    "fmt"
)

// json

type student struct {
    Name string `json:"name" db:"database_name" ini:"config_name"`
}
func main() {
    s := student{
        Name: "小明",
    }
    js, err := json.Marshal(s)
    if err != nil {
        fmt.Println(err)
    }
    fmt.Println(string(js)) // {"Name":"小明"}
}

JSON转结构体

json.Unmarshal(字节类型的字符串, 要接受的变量地址)
package main

import (
    "encoding/json"
    "fmt"
)

// json
type student struct {
    Name string `json:"name"`
}

func main() {
    var p1 student
    str := "{\"name\":\"小明\"}"
    json.Unmarshal([]byte(str), &p1)
    fmt.Printf("%#v\n", p1) // main.student{Name:"小明"}
}

构造函数

利用构造函数去返回一个person结构体对象

package main

import "fmt"

// 构造函数
type person struct {
    name  string
    age   int
    class string
    aihao []string
}

func newPerson(name string, age int, class string, aihao []string) person {
    return person{
        name:  name,
        age:   age,
        class: class,
        aihao: aihao,
    }
}

func main() {
    var p1 = newPerson("小明", 22, "数学", []string{"电脑", "手机"})
    fmt.Println(p1) // {小明 22 数学 [电脑 手机]}
}

返回结构体指针

package main

import "fmt"

// 构造函数
type person struct {
    name  string
    age   int
    class string
    aihao []string
}

func newPerson(name string, age int, class string, aihao []string) *person {
    return *person{
        name:  name,
        age:   age,
        class: class,
        aihao: aihao,
    }
}

func main() {
    var p1 = newPerson("小明", 22, "数学", []string{"电脑", "手机"})
    fmt.Println(p1) // {小明 22 数学 [电脑 手机]}
}
