方法和接收者

发表于 2023-01-19
  • 余星酒
首页 编程 Golang 基础

方法

Go语言中的方法[Method]是一种作用于特定类型变量的函数【不是随便一个类型就可以调用】

type animal struct {
    name string
    age  int
}

// 构造函数
func newAnimal(name string, age int) animal {
    return animal{
        name: name,
        age:  age,
    }
}

// 方法
func (a animal) jiao() {
    fmt.Printf("%s在叫", a.name)
}

任意类型添加方法

只能给自己定义的类型添加，不能给已有的类型添加

// 错误的
func (i int) hello(){
    fmt.Println("i'm is a int")
}


// 正确的
type myInt int
func (m myInt) hello(){
    fmt.Println("i'm is a int")
}

接收者

这种特定类型变量叫做接收者[Receiver]【接收者的概念相当于其他语言中的this或self】

定义格式

func (接收者 接收者类型) 方法名(参数列表) (返回参数){
    函数体
}

其中

接收者变量 ：接收者中的参数变量名在命名时，官方建议使用接收者类型名的第一个小写字母，而不是self、this之类的命名【Person类型的接收者变量应命名为p，Connector类型的接收者变量应该命名为c等】

接收者类型 ：接收者类型和参数类似，可以是指针类型和非指针类型

方法名、参数列表、返回参数 ：具体格式与函数定义相同

package main

import "fmt"

type animal struct {
    name string
    age  int
}

// 构造函数
func newAnimal(name string, age int) animal {
    return animal{
        name: name,
        age:  age,
    }
}

// 方法
func (a animal) jiao() {
    fmt.Printf("%s在叫", a.name)
}

func main() {
    dog := newAnimal("狗", 11)
    dog.jiao() // 狗在叫
}

值接收者

接收副本，不会改变原来的值

package main

import "fmt"

// 方法

type animal struct {
    name string
    age  int
}

// 构造函数
func newAnimal(name string, age int) animal {
    return animal{
        name: name,
        age:  age,
    }
}

// 方法
func (a animal) jiao() {
    fmt.Printf("%s在叫", a.name)
}

// 值接收
func (a animal) guonian() {
    a.age++
}

func main() {
    dog := newAnimal("狗", 11)
    fmt.Println(dog.age) // 11
    dog.guonian()
    fmt.Println(dog.age) // 11
}

指针接受者

接收内存地址，改变原来的值

package main

import "fmt"

// 方法

type animal struct {
    name string
    age  int
}

// 构造函数
func newAnimal(name string, age int) animal {
    return animal{
        name: name,
        age:  age,
    }
}

// 方法
func (a animal) jiao() {
    fmt.Printf("%s在叫", a.name)
}

// 指针引用
func (a *animal) guodanian() {
    a.age++
}
func main() {
    dog := newAnimal("狗", 11)
    fmt.Println(dog.age) // 11
    dog.guodanian()
    fmt.Println(dog.age) // 12
}

指针接收者应用场景

  1. 需要修改接收者中的值
  2. 接收者是拷贝代价比较大的大对象
  3. 保证一致性，如果有某个方法使用了指针接收者，那么其他的方法也应该使用指针接收者
