配置环境变量

windows下配置

在系统变量中添加以下信息

命令行添加代理设置

go env -w GOPROXY=https://goproxy.cn.direct
go env -w GO111MODULE=on

Linux下配置

cd /etc/profile.d

vi go.sh

if [[ "x" == "x${GROOT}" ]]; then
    export GO111MODULE=on
    export GOPROXY=https://goproxy.cn.direct
    export GOROOT=/usr/local/go
    export GOPATH=${HOME}/go
    export PATH=${GOROOT}/bin:${GOPATH}/bin:${PATH}
fi

VSCode快捷键

设置 —> 首选项 —> 键盘快捷方式

  1. 行注释 ctr+/
  2. 块注释 shift+alt+a
  3. 删除行 ctr+shift+k
  4. 查找文件 ctr+e
  5. 替换文件 ctrl+h
  6. 打开设置命令行 ctr+shift+p
  7. 向上/向下复制行 shift+alt+↓/↑
  8. 将选择添加到下一个查找匹配 ctrl+d
