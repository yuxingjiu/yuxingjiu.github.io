# VSCode 快捷键

设置 —> 首选项 —> 键盘快捷方式

  1. 行注释 ctr+/
  2. 块注释 shift+alt+a
  3. 删除行 ctr+shift+k
  4. 查找文件 ctr+e
  5. 替换文件 ctrl+h
  6. 打开设置命令行 ctr+shift+p
  7. 向上 / 向下复制行 shift+alt+↓/↑
  8. 将选择添加到下一个查找匹配 ctrl+d

# VSCode 设置快捷命令
