# 流程控制

流程控制是每种编程语言控制逻辑走向和执行次序的重要部分，流程控制可以说是一门语言的经脉

Go 语言中最常见的流程控制有 if 和 for，而 switch 和 goto 主要是为了简化代码降低重复代码而生的结构，属于扩展类的流程控制

# for range

# 键值循环

Go 语言中可以使用 for range 遍历数组、切片、字符串、map 及通道 (channel)

通过 for range 遍历的返回值有以下规律

  1. 数组、切片、字符串返回索引和值
  2. map 返回键和值
  3. 通道 (channel) 只返回通道内的值

# goto

跳转到指定标签

goto 语句通过标签进行代码间的无条件跳转，goto 语句可以在快速调处循环、避免重复退出上有一定的帮助

Go 语言中使用的 got 语句能简化一些代码的实现过程，如双层嵌套的 for 循环要退出时

# break

break 只能跳出当前循环，如果存在多层循环时，可以仿造 goto 的写法跳出多层循环

通过 break 后面加参数，可以达到跳出多层循环

# continue

continue 时跳过本次循环，如果不想执行本层循环，但是又不想直接跳出最外面（即实现 break 的功能）
