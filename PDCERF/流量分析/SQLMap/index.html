# SQLMap --os-shell

# 实现条件

  1. root 权限
  2. 目标绝对路径
  3. secure-file-priv= （为空） 这个 MySQL 默认不存在，需要自行添加
  4. php 关闭魔术引号

# 实现过程

  1. 确认后端语言（asp、php、jsp 等）
  2. 发现或者提供一个可写目录（绝对路径）
  3. 上传两个文件 tmpuqbez.php（前者为文件上传脚本） 和 tmpbllmx.php（命令执行脚本）

通过 wireshark 抓取流量查看这两个代码

对传参进行了 url 编码，对文件进行了 16 进制编码

这个 php 文件就是一个文件上传，并将目标文件权限设置为 0755，文件生成利用的 LINES TERMINATED BY ，所以前面会带有之前 sql 执行的结果

查看另一个文件

可以看到先去访问该文件是否存在，返回 404，之后通过文件上传命令执行脚本

之后便是命令执行

最后当退出时，自动删除这两个文件

# 总结

  1. 先通过 SQL 语句上传一个文件上传脚本
  2. 访问命令执行脚本，不存在则上传命令执行脚本
  3. 执行命令
  4. 退出 os-shell 自动删除脚本
