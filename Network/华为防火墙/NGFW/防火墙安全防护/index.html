# web 安全

URL 过滤库

URL 过滤功能可以 <font color="#ff0000"> 对用户访问的 URL 进行控制，允许或禁止用户访问某些网页资源 </font>，达到规范上网行为的目的，当企业用户发起 HTTP 或 HTTPS 的 RUL 请求时，通过 URL 过滤功能可以实现对用户请求进行 < mark style="background: #FFB8EBA6;"> 放行 </mark>、<mark style="background: #FFB8EBA6;"> 告警 </mark > 或 < mark style="background: #FFB8EBA6;"> 阻断 </mark>

# 应用安全

应用协议识别

<font color="#ff0000"> 传统防火墙通过协议和端口号来区别应用程序 </font>，对使用相同协议和端口号的不同应用程序，传统防火墙无法对它们进行有效的区分

<font color="#ff0000">NGFW 通过利用应用程序的特征 </font>，能够精确地识别各种常见的应用程序，识别出来后进行精确地控制（封堵、限流）

# 入侵防御

攻击检测

<font color="#ff0000"> 入侵防御是一种安全机制 （<mark style="background: #ABF7F7A6;"> 既能发现又能阻止入侵行为的新安全防御技术 </mark>）</font>，通过分析网络流量、检测入侵（包括缓冲区溢出功能机、木马、蠕虫等），并通过一定的响应方式，实时地中止入侵行为，保护企业信息系统和网络架构免受伤害

  1. 可以实时阻断攻击 ：设备采用 <mark style="background: #ABF7F7A6;"> 直路方式 </mark > 部署在网络中，能够在检测到入侵时，实时对入侵活动和攻击性流量进行拦截，把其对网络的入侵降到最低
  2. 深层防护 （2-7）：由于新型的攻击都隐藏在 TCP/IP 协议的应用层里，入侵防御能 <mark style="background: #ABF7F7A6;"> 检测报文应用层的内容 </mark>，还可以对网络数据流量组进行协议分析和检测，并根据攻击类型、策略等来确定哪些流量应该被拦截
  3. 全方位防护（客户端 + 服务器）
  4. 内外兼防 ：系统对经过的流量都可以进行检测，既可以对服务器防护，也可以对客户端进行防护
  5. 不断升级，精确防护

# 邮件安全

实时反垃圾邮件

垃圾邮件是指未经用户许可强行发送的电子邮件，邮件包含广告，宣传资料、病毒等内容，对用户来说，垃圾邮件除了会影响正常的邮件阅读，还可能包含病毒等有害信息，对服务提供商来说，垃圾邮件会造成邮件拥塞，降低网络运行效率

内容过滤、关键字过滤

IP 地址检查可以防止垃圾邮件在内网泛滥
邮件内容过滤既可以过滤掉匿名邮件，也可以通过检查邮件内容控制内网用户的邮件发送或接收权限

# 反病毒

附件病毒检查与安全提醒

<font color="#ff0000"> 反病毒是一种安全机制 </font>，可以通过 < mark style="background: #ABF7F7A6;"> 识别 </mark > 和 < mark style="background: #ABF7F7A6;"> 处理病毒文件 </mark > 来保证网络安全，避免由病毒文件而引起的数据破坏、权限更改和系统崩溃等情况发生
反病毒功能可以凭借庞大且不断更新的病毒特征库有效地保护网络安全、防止病毒文件侵害系统数据，将病毒检测设备部署在企业网的入口
