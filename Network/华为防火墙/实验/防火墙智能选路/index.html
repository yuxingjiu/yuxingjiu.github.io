# 策略路由

企业主要分为市场部和研发部两个部门，FW 位于企业网出口，该企业部署了两条接入 Internet 的链路 ISP1、ISP2，ISP1 上网速度快、网络速度稳定当时费用比较高，ISP2 上网费用低廉，但是网络相对慢一些，

  1. 市场部（vlan10）对网速要求比较高，通过 ISP1（联通）访问 Internet
  2. 研发部（vlan20）对网速要求不高，通过 ISP2（电信）访问 Internet
  3. web 业务走 ISP1
  4. P2P 业务走 ISP2
  5. 其他属于联通的业务走联通，属于电信的业务走电信，非联通、电信业务在两条链路上负载
  6. 任意一条 ISP 链路故障后网络不中断
