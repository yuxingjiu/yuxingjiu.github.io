GREVPN

SW1 配置

[SW1]vlan batch 10 20 192
[SW1]interface Ethernet0/0/1
[SW1-Ethernet0/0/1]port link-type access
[SW1-Ethernet0/0/1]port default vlan 10
[SW1]interface Ethernet0/0/2
[SW1-Ethernet0/0/2]port link-type access
[SW1-Ethernet0/0/2]port default vlan 20
[SW1]interface Ethernet0/0/3
[SW1-Ethernet0/0/3]port link-type access
[SW1-Ethernet0/0/3]port default vlan 192
[SW1]interface Vlanif10
[SW1-Vlanif10]ip address 172.16.1.254 255.255.255.0
[SW1]interface Vlanif20
[SW1-Vlanif20]ip address 172.16.2.254 255.255.255.0
[SW1]interface Vlanif192
[SW1-Vlanif192]ip address 192.168.255.1 255.255.255.0
[SW1]ip route-static 0.0.0.0 0 192.168.255.2

ISP 配置

[ISP]interface GigabitEthernet0/0/2
[ISP-GigabitEthernet0/0/2]ip address 202.96.137.1 255.255.255.0
[ISP]interface GigabitEthernet0/0/3
[ISP-GigabitEthernet0/0/3]ip address 113.100.1.1 255.255.255.0

FW1 配置

  1. 接口 IP 配置
[FW1]interface GigabitEthernet1/0/0
[FW1-GigabitEthernet1/0/0]ip address 192.168.255.2 255.255.255.0
[FW1]interface GigabitEthernet1/0/6
[FW1-GigabitEthernet1/0/6]ip address 202.96.137.88 255.255.255.0
  1. 区域划分
[FW1]firewall zone trust
[FW1-zone-trust]add interface GigabitEthernet1/0/0
[FW1]firewall zone untrust
[FW1-zone-untrust]add interface GigabitEthernet1/0/6
[FW1]firewall zone dmz
[FW1-zone-dmz] add interface Tunnel 0
  1. GREVPN 配置
[FW1]interface Tunnel 0
[FW1-Tunnel0]ip address 192.168.254.1 255.255.255.0
[FW1-Tunnel0]tunnel-protocol gre
[FW1-Tunnel0]source 202.96.137.88
[FW1-Tunnel0]destination 113.100.1.99
  1. 路由配置
[FW1]ip route-static 0.0.0.0 0.0.0.0 202.96.137.1
[FW1]ip route-static 172.16.1.0 255.255.255.0 192.168.255.1
[FW1]ip route-static 172.16.2.0 255.255.255.0 192.168.255.1
[FW1]ip route-static 172.17.1.0 255.255.255.0 192.168.254.2
  1. 安全策略配置
[FW1]security-policy
[FW1-policy-security]default action permit

FW2 配置

  1. 接口 IP 配置
[FW2]interface GigabitEthernet1/0/0
[FW2-GigabitEthernet1/0/0]ip address 172.17.1.254 255.255.255.0
[FW2]interface GigabitEthernet1/0/6
[FW2-GigabitEthernet1/0/6]ip address 113.100.1.99 255.255.255.0
  1. 区域划分
[FW2]firewall zone trust
[FW2-zone-trust]add interface GigabitEthernet1/0/0
[FW2]firewall zone untrust
[FW2-zone-untrust]add interface GigabitEthernet1/0/6
[FW2]firewall zone dmz
[FW2-zone-dmz] add interface Tunnel 0
  1. GREVPN 配置
[FW2]interface Tunnel 0
[FW2-Tunnel0]ip address 192.168.254.1 255.255.255.0
[FW2-Tunnel0]tunnel-protocol gre
[FW2-Tunnel0]source 113.100.1.99
[FW2-Tunnel0]destination 202.96.137.88
  1. 路由配置
[FW2]ip route-static 0.0.0.0 0.0.0.0 113.100.1.1
[FW2]ip route-static 172.16.1.0 255.255.255.0 192.168.254.1
  1. 安全策略配置
[FW2]security-policy
[FW2-policy-security]default action permit

详细安全策略配置

先 ping 一下，查看防火墙的 session table

从 session 表中可以看到存在 gre、icmp 分别对应的正向会话信息（也存在反向会话）

GRE => untrust -> Local & Local -> untrust
ICMP => trust -> dmz & dmz -> trust

FW1 安全策略配置

[FW1]security-policy
[FW1-policy-security-rule-yw]default action deny

[FW1-policy-security]rule name gre
[FW1-policy-security-rule-gre]source-zone untrust
[FW1-policy-security-rule-gre]destination-zone local
[FW1-policy-security-rule-gre]source-zone local
[FW1-policy-security-rule-gre]destination-zone untrust
[FW1-policy-security-rule-gre]source-address 113.100.1.99 mask 255.255.255.255
[FW1-policy-security-rule-gre]source-address 202.96.137.88 mask 255.255.255.255
[FW1-policy-security-rule-gre]destination-address 113.100.1.99 mask 255.255.255.255
[FW1-policy-security-rule-gre]destination-address 202.96.137.88 mask 255.255.255.255
[FW1-policy-security-rule-gre]action permit

[FW1-policy-security]rule name yw
[FW1-policy-security-rule-yw]source-zone trust
[FW1-policy-security-rule-yw]destination-zone dmz
[FW1-policy-security-rule-yw]source-zone dmz
[FW1-policy-security-rule-yw]destination-zone trust
[FW1-policy-security-rule-yw]source-address 172.16.1.0 24
[FW1-policy-security-rule-yw]source-address 172.17.1.0 24
[FW1-policy-security-rule-yw]destination-address 172.16.1.100 32
[FW1-policy-security-rule-yw]destination-address 172.17.1.10 32
[FW1-policy-security-rule-yw]action permit

命令行配置

基础配置

  1. FW1 配置
[FW1]int g1/0/0
[FW1-GigabitEthernet1/0/0]ip a 192.168.255.2 24
[FW1]int g1/0/1
[FW1-GigabitEthernet1/0/1]ip a 202.96.137.88 24
[FW1]int tunnel 0
[FW1-Tunnel0]ip a 10.1.1.1 24
[FW1-Tunnel0]tunnel-protocol gre
[FW1-Tunnel0]source 202.96.137.88
[FW1-Tunnel0]des 113.100.1.99
[FW1-zone-untrust]fire zone trust
[FW1-zone-trust]add int g1/0/0
[FW1]fire zone untrust
[FW1-zone-untrust]add int g1/0/1
[FW1]fire zone dmz
[FW1-zone-dmz]add int tunnel 0
[FW1]ip route-static 0.0.0.0 0 202.96.137.1
[FW1]ip route-static 172.16.1.0 24 192.168.255.1
[FW1]ip route-static 172.16.2.0 24 192.168.255.1
[FW1]ip route-static 172.17.1.0 24 10.1.1.2
[FW1]security-policy
[FW2-policy -security]default action permit
  1. FW2 配置
[FW2]int g1/0/1
[FW2-GigabitEthernet1/0/1]ip a 113.100.1.99 24
[FW2]int g1/0/2
[FW2-GigabitEthernet1/0/2]ip a 172.17.1.1 24
[FW2-GigabitEthernet1/0/2]int tunnel 0
[FW2-Tunnel0]ip a 10.1.1.2 24
[FW2-Tunnel0]tunnel-protocol gre
[FW2-Tunnel0]source 113.100.1.99
[FW2-Tunnel0]des 202.96.137.88
[FW2]fire zone trust
[FW2-zone-trust]add int g1/0/2
[FW2-zone-trust]fire zone untrust
[FW2-zone-untrust]add int g1/0/1
[FW2-zone-untrust]fire zone dmz
[FW2-zone-dmz]add int tunnel 0
[FW2]ip route-static 0.0.0.0 0 113.100.1.1
[FW2]ip route-static 172.16.2.0 24 10.1.1.1
[FW2]security-policy
[FW2-policy-security]default action permit

查看 session 会话表

从 session 表中可以看到存在 gre、http 分别对应的正向会话信息（也存在反向会话）

GRE => untrust <——> Local & 113.100.1.99 <——> 202.96.137.88
ICMP => trust <——> dmz & 172.16.1.0 <——> 172.17.1.0
