# 访问内网服务

内网 server3 为 WWW 服务器，部署 NAT-server，对外开发 80 端口，使用外网 client3 测试。

​​

虽然外网可以访问，但是内网却无法访问，因为配置了 http、https 等一系列业务 策略路由要走电信链路，所以这里如果还想实现访问内网服务器，可以得额外在添加一条策略路由将去往 172.16.1.0/24 网段的不做策略路由

​​

这样就可以正常访问了
