基础指令

[!info] 关闭提示信息
[SW1]undo info-center enable

[!info] 用户视图切换系统视图
system-view

[!info] 系统视图切换接口视图
[SW1]interface Ethernet0/0/1

[SW1]interface GigabitEthernet0/0/1

[!info] 设备重命名

[SW1]sysname SW1

信息查看

[!info] 查看所有配置信息

[SW1]display current-configuration

[!info] 查看当前视图信息

[SW1]display this
[SW1-GigabitEthernet0/0/1]display this
[SW1-Ethernet0/0/1]display this

[!info] 查看指定接口MAC

[SW1]display interface Ethernet 0/0/1

[!info] 查看指定接口IP信息

[R1]display ip interface GigabitEthernet 0/0/1

[!info] 查看接口IP信息

[R1]display ip interface brief

[!info] 查看MAC表信息

[SW1]display mac-address

用户相关

基于密码认证

  1. 进入虚拟终端
[AR1]user-interface vty 0 4
  1. 设置认证模式
[AR1-ui-vty0-4]authentication-mode password
  1. 设置口令
[AR1-ui-vty0-4]set authentication password simple 123.com
  1. 设置权限
[AR1-ui-vty0-4]user privilege level 15

基于用户和密码认证

  1. 进入3a认证模式
[AR1]aaa
  1. 设置用户密码
[AR1-aaa]local-user admin password cipher 123.com
  1. 设置用户等级
[AR1-aaa]local-user admin privilege level 15
  1. 设置用户类型
[AR1-aaa]local-user admin service-type telnet
  1. 进入虚拟终端
[AR1]user-interface vty 0 4
  1. 设置基于用户密码模式
[AR1-ui-vty0-4]authentication-mode aaa
  1. 指定哪种协议可以访问
[AR1-ui-vty0-4]protocol inbound telnet
  1. 设置用户权限（这里的和aaa模式里面的权限一样）
[AR1-ui-vty0-4]user privilege level 15

路由命令

[!info] 配置静态路由

[R1]ip route-static 目标网段 掩码 下一跳IP
[R1]ip route-static 10.1.1.0 24 192.168.1.2

[!info] 配置Router-id

[R1]ospf 1 router-id ?
X.X.X.X OSPF Private router ID value

[!info] 配置Area

[R1-ospf-1]area ?
INTEGER<0-4294967295> OSPF area ID (Integer)
X.X.X.X OSPF area ID (IP address)
[R1-ospf-1]area 0 （area 0是骨干区域，必须配置的）

[!info] 配置OSPF路由

[R1-ospf-1-area-0.0.0.0]network 当前路由器接口的网段 反码
[R1-ospf-1-area-0.0.0.0]network 10.1.1.0 0.0.0.255

[!info] 查看邻居信息

[R1]display ospf peer

[!info] 查看邻居表

[R1]display ospf peer brief

[!info] 查看LSDB表

[R1]display ospf lsdb

[!info] 查看OSPF路由表

[R1]display ospf routing

VLAN命令

[!info] 创建vlan

[SW1]vlan 10

[SW1]vlan batch 20 30

[SW1]vlan batch 30 to 40

[!info] 删除vlan

[SW1]undo vlan 10

[SW1]undo vlan batch 20 30

[!info] 查看vlan

[SW1]display vlan

[!info] 查看允许通过列表

[SW1]display port vlan
设置接口类型

[SW1-Ethernet0/0/1]port link-type ?
  access        Access port
  dot1q-tunnel  QinQ port
  hybrid        Hybrid port
  trunk         Trunk port

[!info] access接口默认vlan id

[SW1-Ethernet0/0/1]port link-type access

[SW1-Ethernet0/0/1]port default vlan 10

[!info] trunk接口pvid

[SW1-Ethernet0/0/1]port link-type trunk

[SW1-Ethernet0/0/1]port trunk pvid vlan 10

[!info] trunk接口允许通过列表

[SW1-Ethernet0/0/1]port trunk allow-pass vlan 3 // 设置一个vlan

[SW1-Ethernet0/0/1]port trunk allow-pass vlan 5 8 9 // 设置多个vlan

[SW1-Ethernet0/0/1]port trunk allow-pass vlan 10 to 20 // 设置一段vlan

[SW1-Ethernet0/0/1]port trunk allow-pass vlan all // 设置所有vlan可以通过

[!info] hybrid接口pvid

[SW1-Ethernet0/0/1]port link-type hybrid

[SW1-Ethernet0/0/1]port hybrid pvid vlan 10

[!info] hybrid接口设置tagged列表

[SW1-Ethernet0/0/1]port hybrid tagged vlan 10 20

[!info] hybrid接口设置untagged列表

[SW1-Ethernet0/0/1]port hybrid untagged vlan 10 20

生成树命令

[!info] 查看stp

[SW1]display stp

[!info] 查看端口类型

[SW1]display stp brief

更改生成树模式

[SW1]stp mode ?
  mstp  Multiple Spanning Tree Protocol (MSTP) mode
  rstp  Rapid Spanning Tree Protocol (RSTP) mode
  stp   Spanning Tree Protocol (STP) mode

配置交换机优先级

[SW1]stp priority ?
  INTEGER<0-61440>  Bridge priority, in steps of 4096

修改系统开销标准

[SW1]stp pathcost-standard ?
  dot1d-1998  IEEE 802.1D-1998
  dot1t       IEEE 802.1T
  legacy      Legacy

修改接口开销值

[SW1-Ethernet0/0/1]stp cost ?
  INTEGER<1-200000000>  Port path cost

修改接口优先级

[SW1-Ethernet0/0/1]stp port priority ?
  INTEGER<0-240>  Port priority, in steps of 16

链路聚合命令

设置优先级

[SW1]lacp priority ? （0为最优）
  INTEGER<0-65535>  Priority value, the default value is 32768

创建聚合口

[SW1]interface Eth-Trunk ?
  <0-63>  Eth-Trunk interface number

设置lacp模式

[SW1-Eth-Trunk1]mode ?（选择lacp-static模式）
  lacp-static  Static working mode
  manual       Manual working mode

[!info] 添加成员

[SW1-Eth-Trunk1]trunkport GigabitEthernet 0/0/1 to 0/0/3

设置最大连接数

[SW1-Eth-Trunk1]max active-linknumber ?
  INTEGER<1-8>  Value of max active linknumber

[!info] 开启抢占模式

[SW1-Eth-Trunk1]lacp preempt enable

设置抢占时间

[SW1-Eth-Trunk1]lacp preempt delay ?
  INTEGER<10-180>  Value of preemption delay time, the default value is 30 (insecond)
[SW1-Eth-Trunk1]lacp preempt delay 10

查看聚合口

[SW1]display eth-trunk ?
  INTEGER<0-63>  Trunk ID
  |              Matching output
  <cr>

DHCP命令

[!info] 查看地址池

[AR1]display ip pool

[!info] 查看详细地址池信息

[AR1]display ip pool name 192_168_1

配置全局DHCP

[!info] 开启DHCP

[AR1]dhcp enable

[!info] 创建地址池

[AR1]ip pool 192_168_1

[!info] 设置地址池范围

[AR1-ip-pool-192_168_1]network 192.168.1.0 mask 24

[!info] 设置网关

[AR1-ip-pool-192_168_1]gateway-list 192.168.1.254

[!info] 设置DNS

[AR1-ip-pool-192_168_1]dns-list 114.114.114.114

[!info] 设置超时时间
[AR1-ip-pool-192_168_1]lease ?

day Day, from 0 to 999

unlimited Unlimited // 不会超时，无限使用

[!info] 设置禁止使用IP

[AR1-ip-pool-192_168_1]excluded-ip-address 192.168.1.1 192.168.1.3

[!info] 接口配置全局DHCP

[AR1]int g0/0/0

[AR1-GigabitEthernet0/0/0]ip address 192.168.1.254 24

[AR1-GigabitEthernet0/0/0]dhcp select global

配置接口DHCP

[!info] 开启DHCP

[AR1]dhcp enable

[!info] 接口配置DHCP（和全局配置命令一样）

[AR1]int g0/0/1

[AR1-GigabitEthernet0/0/1]ip address 192.168.2.254 24

[AR1-GigabitEthernet0/0/1]dhcp select interface

[AR1-GigabitEthernet0/0/1]dhcp server dns-list 114.114.114.114

[AR1-GigabitEthernet0/0/1]dhcp server excluded-ip-address 192.168.1.1 192.168.1.3

配置中继DHCP

[!info] 开启DHCP

[R2]dhcp enable

[!info] 在对应网段接口中配置中继

[R2]int e0/0/0

[R2-Ethernet0/0/0]ip address 192.168.1.254 24

[R2-Ethernet0/0/0]dhcp select relay

[R2-Ethernet0/0/0]dhcp relay server-ip 10.1.1.1 // 配置中继DHCP对应的DHCP服务接口IP

[R2]int e0/0/1

[R2-Ethernet0/0/1]ip address 192.168.2.254 24

[R2-Ethernet0/0/1]dhcp select relay

[R2-Ethernet0/0/1]dhcp relay server-ip 10.1.1.1 // 配置中继DHCP对应的DHCP服务接口IP

ACL命令

基本ACL配置

[!info] 进入基本ACL配置

[AR1]acl number 2000 （2000 ~ 2999）

[!info] 允许某网段（IP）通过

[AR1-acl-basic-2000]rule permit source IP段 对应反码

[AR1-acl-basic-2000]rule permit source 10.1.1.0 0.0.0.255

[AR1-acl-basic-2000]rule permit source 具体IP 0

[AR1-acl-basic-2000]rule permit source 10.1.1.1 0

[!info] 拒绝某网段（IP）通过

[AR1-acl-basic-2000]rule deny source IP段 对应反码

[AR1-acl-basic-2000]rule deny source 10.1.1.0 0.0.0.255

[AR1-acl-basic-2000]rule deny source 具体IP 0

[AR1-acl-basic-2000]rule deny source 10.1.1.1 0

高级ACL配置

[!info] 进入高级ACL配置

[AR1]acl number 3000 （3000 ~ 3999）

允许或拒绝IP访问网段（或网段某台主机）

  1. deny ：拒绝
  2. permit ：允许
[AR1-acl-adv-3000]rule deny/permit 协议 source IP 反码 destination IP 反码
[AR1-acl-adv-3000]rule deny/permit 协议 source IP段  0 destination IP段  0

[AR1-acl-adv-3000]rule deny ip source 10.1.1.0 0.0.0.255 destination 10.1.2.0 0.0.0.255
[AR1-acl-adv-3000]rule deny ip source 10.1.1.2 0 destination 10.1.2.2 0

允许或拒绝IP访问网段某台主机端口

  1. eq ：等于
  2. gt ：大于等于
  3. lt ：小于等于
[AR1-acl-adv-3000]rule deny/permit 协议 source IP 反码 destination IP 反码 destination-port 符号 端口（或端口协议）
[AR1-acl-adv-3000]rule deny ip source 10.1.1.0 0.0.0.255 destination 10.1.2.0 0.0.0.255 destination-port eq 21

[AR1-acl-adv-3000]rule deny/permit 协议 source IP段  0 destination IP段  0 destination-port 符号 端口（或端口协议）
[AR1-acl-adv-3000]rule deny ip source 10.1.1.2 0 destination 10.1.2.2 0 destination-port eq 21

在接口生效策略
分清楚数据的传输方向

  1. inbound ：入站
  2. outbound ：出站
[AR1-GigabitEthernet0/0/2]traffic-filter outbound acl 2000
[AR1-GigabitEthernet0/0/0]traffic-filter inbound acl 3000

PPPoE命令

PPPoE服务器端配置

创建配置地址池

[AR2]ip pool 200_1_1 
[AR2-ip-pool-200_1_1]network 200.1.1.0 mask 24
[AR2-ip-pool-200_1_1]gateway-list 200.1.1.1
[AR2-ip-pool-200_1_1]dns-list 114.114.114.114

创建虚拟接口

[AR2]interface Virtual-Template 1 // 创建虚拟接口

指定认证模式

[AR2-Virtual-Template1]ppp authentication-mode chap

设置对端地址从本地地址池给定

[AR2-Virtual-Template1]remote address pool 200_1_1

设置网关

[AR2-Virtual-Template1]ip address 200.1.1.1 24

绑定物理口

[AR2]int g0/0/0	
[AR2-GigabitEthernet0/0/0]pppoe-server bind virtual-template 1

设置账号密码

[AR2]aaa	
[AR2-aaa]local-user admin password cipher 123.com
[AR2-aaa]local-user admin service-type ppp

PPPoE客户端配置

创建配置拨号口

[AR1]interface Dialer 1
[AR1-Dialer1]link-protocol ppp // 设置协议类型
[AR1-Dialer1]dialer user admin // 使用者
[AR1-Dialer1]dialer bundle 1 // 绑定物理编号
[AR1-Dialer1]ppp chap user admin // 认证用户
[AR1-Dialer1]ppp chap password cipher 123.com // 用户密码
[AR1-Dialer1]ip address ppp-negotiate // 地址从对端获取

绑定物理口

[AR1]interface GigabitEthernet 0/0/0
[AR1-GigabitEthernet0/0/0]pppoe-client dial-bundle-number 1 // 绑定物理口

配置路由

[AR1]ip route-static 0.0.0.0 0 Dialer 1 // 指定出口路由

配置策略进行NAT转换

[AR1]acl number 2000
[AR1-acl-basic-2000]rule permit source any 
[AR1]interface Dialer 1
[AR1-Dialer1]nat outbound 2000
