# 网络通讯

指的是终端设备之间通过计算机网络进行的通讯

<font color="#00b050"> 终端 </font> ： 手机、电脑、平板

<font color="#00b050"> 传输媒介 </font> ：网线、光纤

<font color="#00b050"> 网络设备 </font> ： 交换机、防火墙

# 网络设备

# 交换机（switch）

交换机是二层设备，指距离终端用户最近的设备，用于终端用户接入网络对数据帧进行交换等

<font color="#00b050"> 为什么交换机每个接口都可以转发数据？</font>
就是因为每个接口都是一个冲突域，自己转发自己的互不干扰

  1. 交换机的每个接口属于一个冲突域
  2. 交换机属于广播域

# 路由器（router）

路由器属于网络层设备（三层设备），可以在因特网中进行报文转发，选择一条合适的路径，将报文传递到下一个路由器或目的地路径中，最后的路由器负责将报文送交目的主机

路由器功能

  1. 跨网段通信（一个接口是一个广播域）
  2. 隔离广播域
  3. 维护路由表运行路由协议
  4. 路径的选择，IP 报文转发
  5. 广域网接入，网络地址转换
  6. 连接通过交换机组建的二层网络

# 网关

网关是一个设备，具体化就是一个 IP 地址，配置在路由器的接口上，报文转发过程中，首先需要确定路径以及前往目的网段的接口，如果目的主机与源知己不在同一网段（网间通信）报文需要先转发到网关，通过网关将报文转发到目的网段（网关地址和本网段主机地址处于统一网络）

# 防火墙（firewall）

防火墙是网络层设备（三层设备），用于控制两个网络之间的安全通信，通过限制、监测、更改跨越防火墙的数据流，尽可能对外屏蔽网络内部的信息结构和运行状况，实现对网络的安全保护

防火墙功能

  1. 隔离不同级别的网络
  2. 实现不同级别的网络之间的访问控制（安全策略）
  3. 用户身份认证
  4. 实现远程接入功能
  5. 实现数据加密及虚拟专用网业务
  6. 执行网络地址转换
  7. 其他安全功能

# 无线设备（无线网络组建）

  1. AP
  2. AC+AP（AC 是 AP 的管理端）

<font color="#00b050"> 胖 AP 和瘦 AP 的区别是否在于可以独立工作，胖 AP 可以独立工作，瘦 AP 需要 AC 控制 </font>

# 网络类型

  1. 局域网 LAN （家庭、学校、公司网络等）
  2. 城域网 MAN （宽带、教育网络等）
  3. 广域网 WAN （因特网）

# 网络拓扑


其中，全网状网络具有最高的可靠性

# 网络架构

  1. 接入层

可以放大节点，指网络中面向用户连接的部分接入层交换机（具有低成本和高端口密度特性）

  1. 汇聚层

主要进行策略实施、流量过滤、汇聚接入层减轻核心层负担

  1. 核心层

实现骨干网络之间的优化传输（所有流量的最终承受者和汇聚者）

# 通信关系

# 网内通信（二层通信）

​同一个网络内部通信，网络位相同，主机位不同的两个 IP 地址​
比如 ：<font color="#00b050">192.168.1.1/24 和 192.168.1.2/24</font>

# 网间通信（三层通信）

​不同网络之间的通信，网络位不同的两个 IP 地址
比如 ：<font color="#00b050">192.168.1.1/24 和 192.168.2.1/24</font>
