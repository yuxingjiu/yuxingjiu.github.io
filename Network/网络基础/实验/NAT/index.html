路由配置

[SW1]ip route-static 0.0.0.0 0 10.1.1.2
[AR1]ip route-static 0.0.0.0 0 50.50.50.2
[AR1]ip route-static 192.168.1.0 24 10.1.1.1
[AR1]ip route-static 192.168.2.0 24 10.1.1.1

动态NAT配置

[AR1]nat address-group 0 50.50.50.5 50.50.50.6
[AR1]acl number 2000
[AR1-acl-basic-2000]rule permit source 192.168.1.0 0.0.0.255 
[AR1]interface GigabitEthernet0/0/2
[AR1-GigabitEthernet0/0/2]nat outbound 2000 address-group 0 no-pat

测试效果

虽说是动态（这里只设置了两个可用地址，故只返回两个接收数据包），但是毕竟公网IP数目有限，如果一直占着公网IP，其他客户端无法正常访问外网

NAPT配置

[AR1]nat address-group 0 50.50.50.5 50.50.50.6
[AR1]acl number 2000
[AR1-acl-basic-2000]rule permit source 192.168.1.0 0.0.0.255 
[AR1]interface GigabitEthernet0/0/2
[AR1-GigabitEthernet0/0/2]nat outbound 2000 address-group 0

测试效果

Easy IP配置

[AR1]acl number 2000
[AR1-acl-basic-2000]rule permit source 192.168.1.0 0.0.0.255 
[AR1]interface GigabitEthernet0/0/2
[AR1-GigabitEthernet0/0/2]nat outbound 2000

测试效果

NAT Server配置

[AR1]acl number 2000
[AR1-acl-basic-2000]rule permit source 192.168.1.0 0.0.0.255
[AR1]interface GigabitEthernet0/0/1
[AR1-GigabitEthernet0/0/1]nat server protocol tcp global 50.50.50.3 www inside 192.168.2.2 www
[AR1-GigabitEthernet0/0/1]nat outbound 2000
[AR1]interface GigabitEthernet0/0/2
[AR1-GigabitEthernet0/0/2]nat server protocol tcp global 50.50.50.3 www inside 192.168.2.2 www
[AR1-GigabitEthernet0/0/2]nat outbound 2000

测试效果（内网服务器端口映射到公网IP端口，公网通过该IP访问，内网也可以通过该IP访问）
