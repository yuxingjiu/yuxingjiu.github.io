DHCP配置

基于全局配置

[AR1]int g0/0/0
[AR1-GigabitEthernet0/0/0]ip address 192.168.1.254 24
[AR1-GigabitEthernet0/0/0]dhcp select global
[AR1]dhcp enable
[AR1]ip pool 192_168_1
[AR1-ip-pool-192_168_1]network 192.168.1.0 mask 24
[AR1-ip-pool-192_168_1]gateway-list 192.168.1.254
[AR1-ip-pool-192_168_1]dns-list 114.114.114.114

基于接口配置

[AR1]int g0/0/1
[AR1-GigabitEthernet0/0/1]ip address 192.168.2.254 24
[AR1-GigabitEthernet0/0/1]dhcp select interface
[AR1-GigabitEthernet0/0/1]dhcp server dns-list 114.114.114.114

中继配置

  1. R1配置
[R1]int e0/0/0	
[R1-Ethernet0/0/0]ip address 10.1.1.1 30
[R1-Ethernet0/0/0]dhcp select global
[R1]dhcp enable 
[R1]ip pool 192_168_1
[R1-ip-pool-192_168_1]network 192.168.1.0 mask 24
[R1-ip-pool-192_168_1]dns-list 114.114.114.114
[R1-ip-pool-192_168_1]gateway-list 192.168.1.254
[R1]ip pool 192_168_2
[R1-ip-pool-192_168_2]network 192.168.2.0 mask 24
[R1-ip-pool-192_168_2]dns-list 114.114.114.114
[R1-ip-pool-192_168_2]gateway-list 192.168.2.254
  1. 中继配置
[R2]dhcp enable 
[R2]int e0/0/0
[R2-Ethernet0/0/0]ip address 192.168.1.254 24
[R2-Ethernet0/0/0]dhcp select relay
[R2-Ethernet0/0/0]dhcp relay server-ip 10.1.1.1
[R2]int e0/0/1
[R2-Ethernet0/0/1]ip address 192.168.2.254 24
[R2-Ethernet0/0/1]dhcp select relay
[R2-Ethernet0/0/1]dhcp relay server-ip 10.1.1.1
[R2]int g0/0/0
[R2-GigabitEthernet0/0/0]ip address 10.1.1.2 30
