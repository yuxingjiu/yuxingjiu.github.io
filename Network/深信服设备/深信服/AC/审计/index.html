# 行为审计

需求背景

  1. 明确责任人
  2. 检测、记录并保留日志
  3. 采取防范保护措施
  4. 数据分类、备份和加密

# 审计架构

审计架构 ：互联网审计 | 访问网站、web 流量、论坛贴吧微博发帖、web 网盘等行为查询，应用、流量、时长等数据统计
业务审计 ：IM 客户端、邮件客户端、TeamViewer、向日葵远程工具、XShell 等运维工具、网盘客户端、U 盘等附件外发审计
客户端审计 ：web、ftp、smb 业务系统的行为和流量审计

# 审计数据存储

默认存储时间为 180 天，这个时间段也是可以修改的 （ 系统管理 => 日志中心配置 => 内置日志中心日志选项 => 勾选 用自动删除最大保留天数，这里可以自定义保留天数 ）

# 内容审计

背景 ：审计论坛发帖、web 邮箱发送邮件、网页网盘上传附件等通过 http/https 协议外发的内容
