认识AC

设备认识

1U型号 1U 表示 4.45cm

2U型号 2U 表示 4.45x2=8.9cm

版本认识

12.x版本称为 上网行为管理

13.x版本称为 全网行为管理

登录AC

深信服 AC/SG设备，各网口默认的出厂IP为

etho [LAN] : 10.251.251.251/24
eth1 [DMZ] : 10.252.252.252/24

但是如果修改了默认的IP，那么可以通过内置的IP来访问

深信服 AC/SG设备的 LAN 口 （默认eth0为lan）DMZ 口 （默认eth1为dmz）

eth0 [LAN] 128.127.125.253/29
eth1 [DMZ] 128.128.125.253/29

网线连接

  1. 路由模式部署时lan和dmz都有保留地址，网桥模式网桥br0和dmz有保留地址，旁路模式部署时管理口有保留地址
  2. 路由模式出厂默认eth0为lan，eth1为dmz（配置被重新定义过网口，eth2口为lan，eth0位dmz，那么eth2有lan口的保留地址[128.127.125.253/29]，eth0有dmz口保留地址[128.128.125.253/29]，旁路模式和网桥模式部署网口的保留地址同样遵循此原则）

知道接口IP地址

eth0接口，需要在电脑上配置一个10.251.251.0/24网段的地址，打开浏览器输入 https://10.251.251.251 登录设备网关控制台
eth1接口，需要在电脑上配置一个10.252.252.0/24网段的地址，打开浏览器输入 https://10.252.252.252 登录设备网关控制台

默认的管理员账号密码为 admin/admin

不知道接口IP地址

即如果设备接口地址被修改，在不知道更改接口后地址的情况下，可以使用内置的IP进行登录

使用同样的方法，访问内置的IP，同样电脑配置同网段IP

eth0接口[LAN]，配置 128.127.125.252/29 访问 128.127.125.252 登录设备
eth1接口[DMZ]，配置 128.128.125.252/29 访问 128.128.125.252 登录设备

sangfor升级工具查找

  1. 安装sangfor升级系统工具的PC和设备交叉线直连
  2. 关闭PC上安装的防火墙及windows自带防火墙

AC恢复出厂设置

控制台

系统管理 => 系统配置 => 配置备份与恢复

sangfor升级工具

在使用这个工具时，先在控制台配置允许远程

系统管理 => 高级配置 => 远程维护 => 启用远程维护，点击启用升级客户端

输入IP和管理员密码后，进入该软件，此时按 F10 ，调出技术支持工具，点击左上角升级 => 恢复出厂设置，找到对应的版本 [版本必须匹配，可以去 https://bbs.singfor.com.cn 进行查找]

交叉线短接设备两个电口

无法登录设备，无法获得设备接口地址

  1. 先将设备关机
  2. 准备一根交叉线 （交叉线就是平时用的网线，只是线序不一致）
  3. 使用交叉线连接设备面板上任意两个非一组bypass电口（AC12.0.12版本后固定接0-2口恢复出厂设置）
  4. 将设备加电开机，一致等待，直到设备重启（恢复出厂配置成功，大概5到10分钟），此时拔掉短接电口的交叉线
  5. 等设备起来后，通过设备默认地址，默认控制台账号 和密码登录

U盘恢复出厂设置

无法登录设备，无法获得设备接口地址

  1. 新建一个txt文档，将其命名为 reset-cfg.txt，将txt文档拷贝到U盘根目录（U盘格式为FAT32）
  2. AC插入U盘，重启设备
  3. 当设备的LED红灯熄灭后，alram灯闪烁，拔出U盘

U盘恢复出厂配置，需要重启设备

AC恢复控制台密码

如果忘记了admin的密码，导致无法登录设备，又需要保留设备配置（不可恢复出厂设置）

交叉线恢复

  1. 确保电脑和设备可以正常通信，访问 https://ACIP/php/rp.php 页面提示创建文件成功，连接交叉线并重启设备

  1. 准备交叉线短接设备任意两个非一组bypass电口（AC12.0.12版本后固定短接0-2接口）

  2. 手动重启设备，重启过程中，注意官场交叉线短接的两个电口 ACT 灯状态，两个电口 ACT 灯同时闪烁10次以后，说明密码恢复完成，此时可以拔掉短接的交叉线，通过默认的控制台账号密码 admin/admin 登录设备即可 （过程大约3到5分钟）

  3. 如果第三步无法根据网口灯状态判断密码恢复是否成功，持续等到5到10分钟左右，尝试 admin/admin 登录

U盘恢复

  1. 新建一个txt文档，将其命名为 reset-cfg.txt，将txt文档拷贝到U盘根目录（U盘格式为FAT32）
  2. 确保电脑和设备可以正常通信，访问 https://ACIP/php/rp.php 页面提示创建文件成功，连接交叉线并重启设备
  3. AC插入U盘，重启设备
  4. 当设备的LED红灯熄灭后，alram灯闪烁，拔出U盘

U盘恢复出厂配置，需要重启设备
