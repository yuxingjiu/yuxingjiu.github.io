用户设置

定义时间段 + 流量管控

定义时间段

保障通道

限制通道

开启空闲允许突破阈值

设置禁止流媒体

4. 要求审计qq聊天，防止核心内容外泄。

审计 + 准入插件 + 准入策略

定义QQ聊天审计

策略管理 => 上网策略 => 新增上网审计策略

定义SSL解密

策略管理 => 上网策略 => 新增上网权限 => SSL管理 => SSL内容识别

选择准入插件解密
